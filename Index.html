How to Set Up GitHub

1. Install Git

Download and install Git from https://git-scm.com/. Once installed, configure Git with your name and email:

git config --global user.name "Your Name"

This command sets your name for any commits you create on your computer.

git config --global user.email "you@example.com"

This command sets your email for commits, linking them to your GitHub account.

2. Create a GitHub Account

Go to GitHub and sign up for a free account if you don't already have one.

3. Set Up SSH or HTTPS for Authentication

To connect to GitHub securely, you can either use SSH keys or HTTPS. Follow the official GitHub SSH guide to generate an SSH key or clone repositories using HTTPS.

GitHub Commands

Common GitHub Commands

git init

Initializes a new Git repository in your project directory.

git clone [URL]

Copies an existing repository from GitHub to your local machine.

git add .

Stages all the changes in your working directory for the next commit.

git commit -m "Commit message"

Commits your staged changes to the repository with a descriptive message.

git push

Uploads your local commits to the remote repository on GitHub.

git pull

Fetches the latest changes from the remote repository and merges them into your current branch.

git status

Displays the status of your working directory, showing any changes that need to be committed.

Working with Branches

Branches allow you to work on different features separately without affecting the main project.

git branch

Lists all branches in your repository and highlights the current branch.

git branch [branch-name]

Creates a new branch with the name you specify.

git checkout [branch-name]

Switches to the specified branch.

git merge [branch-name]

Merges the changes from the specified branch into your current branch.

git branch -d [branch-name]

Deletes the specified branch.
